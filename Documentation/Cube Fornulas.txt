 1) Rune Upgrades
------------------
    Any 3 Runes up to Lem   = Upgraded to Next Rune Type
    Any 2 Runes from Pul up = Upgraded to Next Rune Type



 2) 3 Magic Charms of Same Type = New magic Charm of Same type
----------------------------------------------------------------
    Any 3 Magic Small Charms  = Rerolled Small Charm (at 100% Character Level)
    Any 3 Magic Large Charms  = Rerolled Large Charm (at 100% Character Level)
    Any 3 Magic Grand Charms  = Rerolled Grand Charm (at 100% Character Level)



 3) 3 Jewels of Same Type = New Jewel of Same type
---------------------------------------------------------------
    Any 3 Magic Jewels = Rerolled Magic Jewel (at 100% Character Level)
    Any 3 Rare Jewels  = Rerolled Rare Jewel  (at 100% Character Level)



 4) 3 Rings = New Ring 
----------------------------------
    Any 3 Magic Rings = Rerolled Magic Ring  (at 75%  Character Level)
    Any 3 Rare Rings  = Rerolled Rare Ring   (at 100% Character Level)



 5) 3 Amulets = New Amulet 
------------------------------------- 
    Any 3 Magic Amulets = Rerolled Magic Amulet (at 75%  Character Level)
    Any 3 Rare Amulets  = rerolled Rare Amulet  (at 100% Character Level)



 6) Unique Arrows & Bolts
--------------------------    
    Quiver of Arrows + Tir Rune  + Regular Amethyst  = Normal Unique Arrows (lvl 6 rquired)**
    Quiver of Bolts  + Tir Rune  + Regular Amethyst  = Normal Unique Bolts (lvl 6 rquired)**
    Quiver of Arrows + Thul Rune + Perfect Topaz     = Exceptional Unique Arrows (lvl 35 rquired)**
    Quiver of Bolts  + Thul Rune + Perfect Topaz     = Exceptional Unique Bolts (lvl 35 rquired)**
    Quiver of Arrows + Fal Rune  + Perfect Saphire   = Elite Unique Arrows (lvl 65 rquired)**
    Quiver of Bolts  + Fal Rune  + Perfect Saphire   = Elite Unique Bolts (lvl 65 rquired)**
    Any Unique Arrows or Bolts + Any Health Potion   = Replenish Arrow Stack to 244**

------------------------------------------------
   Set & Unique Weapon/Armor Types/Shield + Lem  Rune   = Adds 1 Socket to Item**
